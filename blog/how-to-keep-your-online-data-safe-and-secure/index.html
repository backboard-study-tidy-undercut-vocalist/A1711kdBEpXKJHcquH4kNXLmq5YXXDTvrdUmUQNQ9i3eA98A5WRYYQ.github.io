How to Keep Your Online Data Safe and Secure

All the recommendations are written with privacy and security in mind. These service providers respect users' privacy & confidentiality and keep Your data safe.

Note: Views are my own, I'll not earn any commission from recommended service provider.


✔ use password manager, preferably Bitwarden

✔ use strong password everywhere (not your name, dob, pet, guessable word; use passphrase)

✔ use different password everywhere

✔ use two factor authentication wherever posible

✔ use FIDO2 U2F security key(like Yubikey) as 2fa, wherever possible

✔ do not click random attachment or link received via email, sms or via any messaging app

✔ do not install random apps from unknown source

✔ use open-source service

✔ keep updated all your tools, apps and OS

✔ avoid using public wi-fi

✔ use encrypted email service, like Tuta Mail

✔ use secure cloud service, like proton drive, mega, pcloud, sync

✔ use vpn service like mullvad vpn(most secure), proton vpn

✔ use secure browser, like firefox, brave, tor(safest)


Pro Tips

✔ degoogle Your services

✔ do not use social media, like fb, X, insta etc

✔ encrypt your data before uploaad on cloud, using cryptomator, veracrypt etc

✔ use LineageOS

✔ use Tor for sensitive work

✔ use tails live os for sensitive work
